# MCMI 2026 Workshop Website

Official website for the 2nd International Workshop on Multi- and Cross-Modal Information for Enhanced Pattern Recognition (MCMI 2026), co-located with ICPR 2026 in Lyon, France.

## Local Development

Run a local server to preview the website:
```bash
python3 -m http.server 8000
```
Visit http://localhost:8000

## Key Files

- **TODOs.md** - Pending content updates
- **js/config.js** - Site configuration (dates, links)
- **js/data.js** - Structured data (organizers, topics, news)
- **css/custom.css** - Custom styling

## Common Updates

- Important dates: `js/config.js`
- Submission link: `js/config.js`
- News items: `js/data.js`
- Organizers: `js/data.js`
- Workshop program: `workshop.html`
